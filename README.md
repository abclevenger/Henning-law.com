# Henning Law Firm PLLC

U.S. Immigration & Visa for German and International Clients.  
[https://henning-law.com](https://henning-law.com) | [henning-law-com.vercel.app](https://henning-law-com.vercel.app)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Stack

Next.js (App Router), React, TypeScript.
